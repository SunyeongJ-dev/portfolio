# Design System

## Philosophy

This portfolio uses a blue-violet dark theme inspired by modern IDEs and code editors.

Design goals:

- Modern and professional
- Developer / programming aesthetic
- High readability
- Minimal scrolling
- Dashboard-style navigation

## Color Rules

- Never use pure `#000000` or `#FFFFFF`.
- Backgrounds should maintain a subtle blue-violet undertone.
- Accent colors should stay within the indigo-violet family.
- Use the mint accent sparingly for badges, tags, and highlights.

---

## Typography

- Prefer clean, highly readable fonts for body text.
- Use monospace fonts selectively to reinforce the programming aesthetic.
- Prioritize readability over stylistic choices.
